Perform a git commit and push:

1. Run `git status` to check what has changed
2. Check that no credentials or secrets (config.json, .env files) are being committed
3. Stage all relevant changes with `git add`
4. Create a commit with a clear, descriptive message in English
5. Push to the remote repository (`git push`)
6. Confirm to me what was committed and pushed, including the commit message
