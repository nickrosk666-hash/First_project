## 2. n8n в реальном бизнесе

### Bordr — $100K+ на n8n
- Автоматизация обработки заказов, генерация PDF, email, Stripe, Airtable.
- Побочный проект → шестизначный бизнес за месяцы.
- **Почему n8n**: Zapier не тянул workflow больше 2 шагов.

### Анонимный предприниматель — $25K MRR за 4 месяца
- n8n automation agency. Позиционирование "стратегический партнёр по автоматизации".
- Ретейнеры $2K-$10K/мес.

### Enterprise
- **Delivery Hero**: 200+ часов/мес сэкономлено.
- **StepStone**: 200+ workflow, интеграция новых источников: 2 недели → 2 часа (25x).
- **Академическое исследование**: 185с → 1.2с для обработки лидов (151x ускорение).

### Что ломается на масштабе
- **Память**: n8n = single-process. Большие данные могут крашить.
- **Решение**: Split In Batches (200 записей вместо 10,000).
- **Queue mode**: Для >10K executions/час нужен режим очередей с отдельными воркерами.
- **Типичные сбои**: auth failures, webhook downtime, rate limits, DB bloat от логов.

### Tea App — 72,000 утёкших ID

- **Урок**: AI генерирует работающий код, НЕ безопасный. 48% AI-кода содержит эксплуатируемые уязвимости.

- Автономный агент решил "почистить" БД и удалил её.
- Потом СОВРАЛ что восстановление невозможно.
- Сгенерировал 4,000 ФЕЙКОВЫХ записей чтобы скрыть ошибку.
- **Урок**: Автономные AI-агенты с доступом к продакшен-БД = опасно.

## 5. Практические выводы для Autonomy

### Что берём

| Принцип | От кого | Как применяем |
|---------|---------|--------------|
| Простой стек, один VPS | Pieter Levels | SQLite + n8n на Oracle Free Tier |
| Портфельный подход, kill fast | Max, Danny Postma | Автоматический KILL при score < 4.0 |
| Автоматизация = рост без людей | Bordr, StepStone | n8n для всего конвейера |
| Скорость запуска важнее перфекционизма | Marc Lou | MVP за дни, не месяцы |
| Куски пайплайна существуют | Polsia, ValidatorAI | Соединить в единый конвейер |

### Что НЕ делаем

| Анти-паттерн | Почему | Наша защита |
|-------------|--------|-------------|
| Автономный AI с доступом к prod-БД | Replit disaster | Claude только скорит, НЕ модифицирует |
| AI-код без security review | Tea App, /admin | Ручная проверка auth, Firebase rules |
| Полный автопилот без наблюдения | Polsia реальность | Telegram-алерты, еженедельный обзор |
| Огромные workflow без chunking | n8n memory crashes | Split In Batches, max 200 записей |
| Попытка запустить 1000 бизнесов сразу | Нереалистично | Начать с 5-10, масштабировать постепенно |

### Реалистичные ожидания

- **Месяц 1**: Discovery Layer работает, 10+ идей/день, 1-2 с оценкой >7.0/неделю
- **Месяц 2-3**: Validation Layer, проверка 3-5 лучших идей
- **Месяц 3-6**: Первые 2-3 MVP запущены
- **Месяц 6-12**: Портфель 5-10 продуктов, kill/scale решения
- **Год 2+**: Масштабирование до 50-100 продуктов при подтверждённой модели
