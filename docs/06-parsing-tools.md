# Инструменты для парсинга и скрапинга

## 1. Встроенные ноды n8n

| Нода | Назначение | Примечания |
|------|-----------|------------|
| **HTTP Request** | Основной инструмент для всех запросов | GET/POST к любым API и сайтам |
| **HTML Node** | Извлечение данных из HTML по CSS-селекторам | Для статических страниц |
| **RSS Feed Trigger** | Опрос RSS-фидов по расписанию | ⚠️ Пропускает старые записи между опросами |
| **RSS Read** | Чтение всех текущих записей из RSS | Лучше для нашего случая — берёт всё |
| **Code Node** | Произвольный JS-код | **Cheerio предустановлен** — jQuery-подобный парсинг HTML без установки |

### Ключевой факт
**Cheerio уже встроен в n8n** — можно парсить HTML прямо в Code-ноде без внешних зависимостей.

## 2. Python-библиотеки (вызов через n8n Execute Command)

| Библиотека | Для чего | Статус 2025-2026 | Рекомендация |

| **Scrapy** | Массовый скрапинг | 39x быстрее BS4 для bulk | ⚠️ Overkill для наших задач |
| **Playwright** | JS-heavy сайты | Поддержка всех браузеров | ⚠️ Только для крайних случаев |

## 5. Антидетект и rate-limiting

### Задержки
- 2-5 секунд рандомизированных пауз между запросами
- n8n: Split In Batches + Wait ноды

### User-Agent ротация
- Ротировать реальные браузерные UA-строки
- Сопровождать правильными заголовками (Accept, Accept-Language)

### IP-ротация
- Webshare free tier для базовых нужд
- Residential прокси для серьёзного скрапинга (платные)

### Вежливый скрапинг
- Уважать robots.txt
- Скрапить в off-peak часы
- Кэшировать ответы
- **Всегда предпочитать RSS/API вместо HTML-скрапинга**

### n8n-специфичное
- Retry on Fail с экспоненциальным backoff
- IF-нода для обработки ошибок
- Хранить seen URLs чтобы не скрапить повторно

## Рекомендуемый стек для Autonomy

```
Tier 1 (основной):
  n8n HTTP Request + RSS Read + Code (Cheerio) — 90% задач

Tier 2 (Python, через Execute Command):
  feedparser — для RSS
  PRAW — для Reddit

Tier 3 (при необходимости):
  n8n Google Trends community node — вместо pytrends
  Playwright — для JS-heavy сайтов
  Webshare прокси — если блокируют
```
