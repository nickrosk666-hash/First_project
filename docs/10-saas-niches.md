# Анализ SaaS-ниш 2025-2026 (данные исследования)

## Главный вывод

> **AI + Vertical SaaS = лучшая ставка.** Vertical SaaS растёт 2-3x быстрее горизонтального. AI — не отдельная ниша, а ускоритель для ВСЕХ ниш. Лучшая комбинация: AI-powered инструмент для конкретной индустрии.

## 2. Доказанные Micro-SaaS ниши для соло-фаундеров

### Обзор рынка Micro-SaaS
- Рынок: **$15.7B (2024) → $59.6B к 2030** (~30% годовой рост)
- **39% независимых SaaS-фаундеров работают соло**
- Типичный срок до прибыльности: **1-2 года**
- Sweet spot: **$5K-50K MRR, 1-2 человека**

## 3. Blue Ocean — Недообслуженные ниши

### Индустрии, которые ВСЁ ЕЩЁ на таблицах

| Индустрия | Текущее состояние | Возможность |
|-----------|-------------------|-------------|
| **Мелкие арендодатели (2-10 объектов)** | Excel, бумага, Venmo | All-in-one management |
| **Строительные SMB** | Бумага, clipboard | Digital twins, трекинг проектов |
| **Специализированные клиники** | Generic EMR | Нишевые workflow для дерматологии, физио |
| **Ресторанный бизнес** | Ручной учёт запасов | Real-time stock + demand forecasting |
| **SMB ESG/compliance** | Ручной трекинг или ничего | Углеродный след, отчётность |
| **Ветклиники** | Устаревший legacy софт | Современный clinic management |
| **Trades (HVAC, pest, electric)** | Clipboard, basic scheduling | Scheduling, pricing, quoting |
| **Коммерческий ландшафтный дизайн** | Ручные расчёты | Estimation, route planning |

### Validation
- Vertical SaaS → **$720B к 2028** (25.89% CAGR)
- Non-tech B2B вертикали растут **250%+ быстрее** (SaaStr)
- Глубокая product stickiness — после adoption очень сложно заменить

## 4. AI Wrapper возможности

### Экономика AI-обёрток

| Метрика | Значение |
|---------|----------|
| Средний доход 1й год | $2,000-$8,000/мес |
| Median выжившие за 1 год | $50K-$200K ARR |
| Top 25% за 1 год | $500K-$1M ARR |
| Net margins (элита) | 35-50% |
| API costs / revenue | 15-30% |
| Достигают >$10K MRR | Только **3-5%** |
| **Failure rate к 2026** | **90%** |

### Что работает как API → Продукт

| API | Wrapper-возможность | Сложность | Moat |
|-----|---------------------|-----------|------|
| Claude/GPT | Нишевые writing tools (legal, medical, grants) | Medium | Высокий с domain templates |
| Whisper | Вертикальная транскрипция (medical, legal, sales) | Low-Med | Средний с vocabulary |
| SD/FLUX | Нишевая генерация (real estate, product photos) | Medium | Средний с fine-tuning |
| Claude/GPT + RAG | Industry knowledge bases (compliance, tax, HR) | Med-High | Высокий с proprietary data |
| Vision APIs | Document extraction (invoices, permits, forms) | Medium | Высокий с training data |

### Ключи к выживанию AI-обёрток
- Gross margins **> 60%**
- Monthly churn **< 5%**
- Switching costs через data lock-in или workflow integration
- Дифференциация **за пределами AI-слоя** (UX, domain knowledge, integrations)
