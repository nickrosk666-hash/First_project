# Конкуренты и аналоги

## Главный вывод: наше конкурентное преимущество

Никто не делает всё вместе. Существующие инструменты закрывают отдельные кусочки:

| Что есть на рынке | Чего НЕТ ни у кого |
|---|---|
| Валидация одной идеи по запросу | **Непрерывный мониторинг** — автоматический поиск каждый день |
| Один источник данных | **Мульти-источник** — Google + Reddit + YouTube + HN + GitHub + новости |
| Тренд-инструменты отдельно, валидаторы отдельно | **Тренд → Идея** — мост между трендом и бизнес-идеей |
| Статичная оценка | **Временной скоринг** — как меняется потенциал идеи со временем |
| Универсальные идеи | **Матчинг под основателя** — подбор под стек и возможности |

### Что все делают одинаково (и плохо)
- Работают **по запросу** — ты вводишь идею, они оценивают
- **Не ищут** идеи сами
- **Один источник** данных (обычно просто LLM)
- Нет **мониторинга** — разовый отчёт

## Стратегические выводы

### Наши 5 уникальных преимуществ (gap analysis)

1. **Непрерывный мониторинг** — все конкуренты работают по запросу, мы — 24/7 автоматически
2. **Мульти-источник** — никто не агрегирует Reddit + Google + YouTube + HN + GitHub + новости
3. **Тренд → Идея** — тренд-инструменты и валидаторы существуют отдельно, никто не соединяет
4. **Временной скоринг** — никто не отслеживает, как потенциал идеи меняется со временем
5. **Zero-cost** — все конкуренты платные ($19-249/мес), мы работаем за ~$1/мес

### Что взять у конкурентов

- **От DimeADozen:** структуру 40-страничного отчёта (когда дойдём до Validation Layer)
- **От BigIdeasDB:** методологию "жалобы → бизнес-идея"
- **От ai-idea-validator:** систему вердиктов KILL/FLIP/BUILD/BET
- **От n8n template #3824:** готовый Reddit-скрапер как стартовую точку
- **От n8n-mcp:** возможность Claude самому строить воркфлоу (фаза 3+)
