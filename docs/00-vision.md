# Видение проекта Autonomy

## Миссия

Построить полностью автономную систему, которая без участия людей:
1. **Находит** перспективные бизнес-идеи в SaaS и digital-секторе
2. **Оценивает** их по объективным критериям
3. **Строит** MVP-продукт
4. **Запускает** на англоязычный рынок
5. **Адаптирует** под обратную связь клиентов

Конечная цель — управление тысячами микро-бизнесов из одного центра.

## Целевой рынок

- **География**: англоязычные страны (US, UK, CA, AU)
- **Сегмент**: SaaS, digital-продукты, автоматизированные сервисы
- **Модель**: подписки (recurring revenue), freemium

## Принципы

1. **Ноль сотрудников** — всё делают агенты (Claude + n8n)
2. **Минимум затрат** — бюджет до $10/мес на инфраструктуру
3. **Данные решают** — каждое решение подкреплено данными, не интуицией
4. **Скорость > Перфекционизм** — быстрый запуск MVP, потом итерации
5. **Масштабируемость** — архитектура рассчитана на 1000+ бизнесов

## Этапы развития

### Фаза 1: Discovery (текущая)
Ежедневный конвейер поиска и оценки бизнес-идей.
Источники: Google → Reddit, YouTube, Facebook → новости, инвест-порталы.

### Фаза 2: Validation
Автоматическая проверка идей: анализ конкурентов, оценка TAM/SAM/SOM,
тестирование спроса через лендинги и рекламу.

### Фаза 3: Build
Автоматическая генерация MVP: код, лендинг, документация.
Шаблоны SaaS-приложений, автоматический деплой.

### Фаза 4: Launch
Автоматический выход на рынок: SEO, контент-маркетинг,
размещение на ProductHunt, Indie Hackers, социальные сети.

### Фаза 5: Operate
Мониторинг метрик, сбор обратной связи, A/B-тесты,
автоматическая адаптация продукта.

### Фаза 6: Scale
Оркестрация портфеля бизнесов, перераспределение ресурсов,
закрытие убыточных, масштабирование прибыльных.

## KPI фазы Discovery

- Минимум 10 новых идей в день
- Минимум 1 идея с оценкой > 7.0 в неделю
- Стоимость Claude API < $3/мес
- Uptime конвейера > 95%
